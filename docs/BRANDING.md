# Branding Notes

Hermey is the Android-native companion client concept for Hermes Agent.

## Positioning

Recommended short description:

> Android-native companion client for Hermes Agent, inspired by Hermex.

Recommended one-liner:

> Hermex brought Hermes Agent to iOS. Hermey brings the same mobile-first operator experience to Android.

## Naming

- Project name: Hermey
- Repository name: hermey-android
- Package namespace candidate: `ai.greymattr.hermey` or `dev.leathal1.hermey`
- Tagline candidate: Hermes in your pocket for Android.

## Affiliation Language

Use:

- Independent Android client for Hermes Agent
- Inspired by Hermex
- Community-built Android companion

Avoid:

- Official Nous Research app
- Official Hermes Agent Android client
- Endorsed by Nous Research
- Maintained by Nous Research

Unless explicit permission or collaboration is established, the project should remain clearly independent.
