## Title
Hermdroid — Hermes Agent for Android

## Short description (80 chars)
Pocket command surface for Hermes Agent: briefs, approvals, and agent control.

## Full description
Hermdroid is the Android-native companion client for Hermes Agent.

- Mobile inbox for agent messages and briefs
- Explicit approval flows for sensitive actions
- Secure settings with encrypted credential storage
- Offline read-only cache for sessions and messages (bbolt)
- Material 3 dynamic color theming

Use Hermdroid to stay connected to your agents from anywhere.
