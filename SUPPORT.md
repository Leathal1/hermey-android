# Support

Hermey is an early-stage open source project.

## Getting Help

Use GitHub issues for:

- Bug reports
- Feature requests
- Documentation gaps
- Architecture discussion
- Android implementation tasks

Please include enough context to make the issue actionable.

## Not Supported Yet

The project does not yet provide:

- Production support
- Stable releases
- Compatibility guarantees
- Official Nous Research support
- Emergency incident response

## Security Issues

Do not open public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).
